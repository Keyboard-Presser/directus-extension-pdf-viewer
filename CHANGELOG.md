# Changelog

All notable changes to `directus-extension-pdf-viewer` are documented here.

## [1.0.0] — 2026-03-22

### Added
- Drag-and-drop PDF upload zone
- Click-to-upload file picker (PDF only)
- Real-time upload progress indicator
- Inline PDF preview via native browser iframe
- Toolbar with open-in-new-tab and remove-file buttons
- Auth-aware asset URL (appends `access_token` for private instances)
- Full Directus theme support via CSS custom properties
- Non-PDF file guard with user-facing error message
- Compatible with Directus `^10.10.0`
